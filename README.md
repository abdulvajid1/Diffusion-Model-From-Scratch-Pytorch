# Diffusion Model From Scratch (PyTorch)

A **from-scratch implementation of Denoising Diffusion Probabilistic Models (DDPM)** using **PyTorch**, built for learning, experimentation, and clarity.

This repository focuses on:
- Understanding diffusion models mathematically & intuitively
- Clean PyTorch implementation (no hidden magic)
- Easy experimentation with noise schedules and training steps
